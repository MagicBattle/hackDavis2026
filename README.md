# Flipit
Flipit is a website to help consumers learn more about what they are eating and what ingredients to look out for. We are inspired to make this project from reading food labels during grocery shopping and wonder whether a particular ingredient is harmful or not. This is a Hackathon Project submission for Hack Davis 2026.

# How we built it
We used React and Tailwind to build the front end of our website. Using Python, we used FastAPI and ClaudeAPI to help read nutrition labels and analyze the ingredients.
